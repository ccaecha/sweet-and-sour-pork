## 📚 背景

### 簡史與起源
咕嚕肉起源於廣東菜，特別是在中國南方地區，當地常用酸甜口味。隨著時間推移，這道菜在20世紀由華人移民帶到西方，逐漸風靡全球。如今，它是世界上最知名、最常見的中式菜餚之一。

![咕嚕肉](img/sweet-and-sour-pork.png)

### 為什麼這道菜受歡迎
咕嚕肉吸引人的地方在於色彩鮮豔、口感層次豐富，以及醋的酸味與糖和番茄醬的甜味之間的平衡。除了美味外，這道菜在農曆新年等節慶期間還有象徵意義，代表著幸福與好運。

### 趣味小知識
- **全球流行：** 在美國、加拿大等地的中餐館，咕嚕肉是最常被點選的菜色之一。
- **地區變化：** 有些版本會用雞肉、豆腐或海鮮取代豬肉，以符合當地口味或飲食需求。
- **象徵意義：** 酸甜口味的和諧象徵中國烹飪中「平衡與統一」的理念。

## 📦 輸入（食材）

### 主要食材

#### 豬肉醃料：

* 500克豬里肌（切塊）
* 1大匙醬油
* 1大匙玉米粉

#### 酸甜醬：

* 1/4杯番茄醬
* 1/4杯米醋（或白醋）
* 1/4杯糖
* 1大匙醬油

#### 炒菜用：

* 1大匙植物油
* 1顆甜椒（切塊）
* 1顆洋蔥（切塊）
* 1/2杯鳳梨（新鮮或罐頭）
* 選用：1根紅蘿蔔（切片）

---

### 🔄 食材替換與變化

本食譜支持彈性替換，以因應飲食限制、個人偏好或食材可得性。

| 原始食材    | 替代選項                        | 備註                           |
| ----------- | ------------------------------- | ------------------------------ |
| 豬里肌      | 雞肉、豆腐、蘑菇、天貝、麵筋    | 適合素食／全素版本             |
| 鳳梨        | 芒果、或省略                    | 芒果增添熱帶風味，甜度不同     |
| 甜椒        | 花椰菜、櫛瓜、甜豆              | 增加色彩與營養                 |
| 白飯        | 藜麥、花椰菜米                  | 低碳或無麩質替代               |

---

### 🌶️ 辣味升級選項

| 選項           | 說明                                 |
| -------------- | ------------------------------------ |
| 加入辣椒       | 炒菜時加入新鮮紅／青辣椒             |
| 使用辣醬       | 拌入酸甜醬中                         |
| 裝飾           | 撒上辣椒片、辣油或切片墨西哥辣椒      |

*如為多人或需顧及飲食敏感者，建議將辣味配料分開提供，保持主菜中性。*

## 🔧 操作步驟

### 前置準備

1. **醃製豬肉**（15–30分鐘）

   * 將豬肉、醬油、玉米粉拌勻。
   * 靜置入味、嫩化。

2. **切菜備料**（10分鐘）

   * 切好甜椒、洋蔥及紅蘿蔔（如用）。
   * 若用罐頭鳳梨，先瀝乾。

---

### 烹調流程

1. **加熱油鍋**

   * 中大火加熱炒鍋或平底鍋。

2. **炸豬肉**（5–7分鐘）

   * 炸至金黃熟透。
   * 用漏勺撈起備用。

3. **炒蔬菜**（3–4分鐘）

   * 加入甜椒、洋蔥、紅蘿蔔。
   * 快炒至脆嫩。

4. **加入醬汁與鳳梨**（2分鐘）

   * 將醬料混合後倒入鍋中，加入鳳梨拌炒。

5. **合併收尾**（2–3分鐘）

   * 將豬肉回鍋，拌炒至醬汁收稠、食材均勻。

---

## 🔎 驗證標準

* 外觀：色澤鮮明、醬汁光亮。
* 口感：豬肉嫩、蔬菜脆。
* 味道：酸甜平衡。
* 選用：豬肉中心溫度約63°C（145°F）。

---

## 🚨 風險與因應

### 風險一：**味道失衡**

* **因應：** 醬汁先試味，適量調整糖或醋。

### 風險二：**豬肉過老**

* **因應：** 遵守烹調時間，炸好後稍微靜置。

### 風險三：**食材短缺**

* **因應：** 採用建議替代品。

---

## 🔁 備案

* **醬汁太稠：** 加水稀釋。
* **醬汁太稀：** 繼續收乾。
* **味道不理想：** 加入醬油或辣醬調整。
* **成品不夠吸引：** 加入更多蔬菜，做成什錦炒菜。

---

## 🍽️ 上桌建議

* **推薦主食：** 蒸香米或白飯
* **健康選擇：** 藜麥或花椰菜米
* **裝飾建議：** 蔥花、芝麻、辣油
* **配菜建議：** 春捲、蛋花湯、涼拌小黃瓜

---

## 🧰 疑難排解

| 現象         | 可能原因   | 解決方法         |
| ------------ | ---------- | ---------------- |
| 豬肉乾柴     | 過度烹調   | 減少炸煮時間     |
| 醬汁太稠     | 粉類過多   | 加水稀釋         |
| 蔬菜軟爛     | 過度加熱   | 大火快炒         |
| 味道平淡     | 調味失衡   | 調整糖或醋比例   |
