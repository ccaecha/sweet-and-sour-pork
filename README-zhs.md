## 📚 背景

### 简要历史与起源
咕噜肉起源于粤菜，尤其是中国南方地区，这里的菜肴常用酸甜口味。随着时间推移，这道菜在20世纪由华人移民带到西方，逐渐风靡全球。如今，咕噜肉已成为世界上最知名、最常见的中餐之一。

![咕噜肉](img/sweet-and-sour-pork.png)

### 这道菜为何受欢迎
咕噜肉以其色彩鲜艳、口感丰富、酸甜平衡而著称。除了美味外，它还具有文化象征意义，尤其在春节等节日，寓意幸福和好运。

### 趣味小知识
- **全球流行：** 咕噜肉是美加等地中餐厅最常点的菜品之一。
- **地区变体：** 有些版本用鸡肉、豆腐或海鲜代替猪肉，以适应不同口味和饮食需求。
- **象征意义：** 酸甜口味的和谐体现了中餐讲究的“平衡与统一”。

## 📦 输入（食材）

### 主要食材

#### 猪肉腌料：

* 500克猪里脊（切块）
* 1汤匙生抽
* 1汤匙玉米淀粉

#### 酸甜酱：

* 1/4杯番茄酱
* 1/4杯米醋（或白醋）
* 1/4杯白糖
* 1汤匙生抽

#### 炒菜用：

* 1汤匙食用油
* 1个彩椒（切块）
* 1个洋葱（切块）
* 1/2杯菠萝（鲜或罐头）
* 可选：1根胡萝卜（切片）

---

### 🔄 食材替换与变体

本食谱支持灵活替换食材，以适应不同饮食需求、口味或现有库存。

| 原始食材   | 替代选项                       | 备注                         |
| ---------- | ------------------------------ | ---------------------------- |
| 猪里脊     | 鸡肉、豆腐、蘑菇、天贝、面筋   | 适合素食/纯素版本            |
| 菠萝       | 芒果、或省略                   | 芒果更热带风味，甜度不同     |
| 彩椒       | 西兰花、节瓜、荷兰豆           | 增加色彩和营养               |
| 白米饭     | 藜麦、花椰菜米                 | 低碳或无麸质替代             |

---

### 🌶️ 辣味增强（可选）

| 选项           | 说明                         |
| -------------- | ---------------------------- |
| 加辣椒         | 炒菜时加入新鲜红/青辣椒      |
| 用辣椒酱       | 酱汁中混入辣椒酱             |
| 装饰           | 撒辣椒圈、辣椒碎或淋辣油     |

*如为多人或需照顾不同口味，建议将辣味配料单独提供，保持主菜中性。*

---

## 🔧 步骤

### 准备工作

1. **腌制猪肉**（15–30分钟）
   * 将猪肉、生抽、淀粉混合，腌制入味。

2. **切菜**（10分钟）
   * 切好彩椒、洋葱、胡萝卜（可选），菠萝沥干。

---

### 烹饪

1. **加热油**
   * 炒锅或平底锅中火加热。

2. **煎猪肉**（5–7分钟）
   * 炸至金黄熟透，用漏勺捞出。

3. **炒蔬菜**（3–4分钟）
   * 加入彩椒、洋葱、胡萝卜，炒至断生。

4. **加酱汁和菠萝**（2分钟）
   * 酱料混合后倒入锅中，加入菠萝翻炒。

5. **合并收汁**（2–3分钟）
   * 放回猪肉，翻炒至酱汁浓稠、食材均匀裹酱。

---

## 🔎 验证标准

* 视觉：色彩鲜艳，酱汁有光泽
* 口感：猪肉嫩滑，蔬菜爽脆
* 味道：酸甜平衡
* 可选：猪肉中心温度约63°C（145°F）

---

## 🚨 风险与应对

### 风险1：**味道失衡**
* **应对：** 酱汁提前试味，适量调整糖或醋。

### 风险2：**猪肉过老**
* **应对：** 严格控制时间，炸好后稍作静置。

### 风险3：**食材缺失**
* **应对：** 参考替换建议。

---

## 🔁 备选方案

* **酱汁太稠：** 加水稀释
* **酱汁太稀：** 继续收汁
* **味道不理想：** 加生抽或辣椒酱调味
* **整体不满意：** 加蔬菜做成杂菜炒，增加酱汁

---

## 🍽️ 上桌建议

* **主食推荐：** 蒸香米饭或白米饭
* **健康选择：** 藜麦或花椰菜米
* **装饰建议：** 葱花、芝麻、辣椒油
* **配菜建议：** 春卷、蛋花汤、拍黄瓜

---

## 🧰 故障排查

| 症状         | 可能原因     | 解决方法         |
| ------------ | ------------ | ---------------- |
| 猪肉发柴     | 炸过头       | 缩短炸制时间     |
| 酱汁太稠     | 淀粉过多     | 加水稀释         |
| 蔬菜发软     | 炒过头       | 大火快炒         |
| 味道寡淡     | 酸甜不平衡   | 调整糖或醋用量   |
